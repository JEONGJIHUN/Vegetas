# Vegetas

**채식 식당을 찾아주는 어플리케이션**

## 기본 환경 설치

<ul>
<li>node@latest</li>
<li>yarn@latest</li>
</ul>

## 패키지 설치

### yarn 설치

```javascript
$ sudo npm install -g yarn
```

### 프로젝트 패키지 설치

```javascript
$ yarn

or

$ npm install
```

1. `root`(VEGETAS), 클라이언트, 서버 폴더 별로 `npm install` or `yarn`으로 설치합니다.

```shell
$ npm install
$ cd vegetas-client
$ npm install
$ cd ../vegetas-server ( 또는 `root`(VEGETAS)에서 cd vegetas-server )
$ npm install
```

2. 각 폴더 별로 npm start 를 실행합니다.
